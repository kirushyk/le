# Le - Machine Learning Framework.

Le is Machine Learning framework written in Pure C. It provides library for:
* Linear Algebra.
* Tensor Manipulation.

Learning algorithms supported at this moment:
* Polynomial Regression.
* Support Vector Machines (SVM).
